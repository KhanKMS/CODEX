1. 강의 목표 : 파이썬 언어를 통해 엑셀 자동화, 웹 스크래핑 자동화 기술 습득 
2. 강의 특징 
   (1) 어려운 파이썬 객체지향언어 문법 설명 보다 코딩 실습 환경 구축 후 
       엑셀 자동화, 웹 스크래핑 등 사무자동 관련 짧은 예제 코딩 실습 우선
   (2) 짧은 코드를 강사와 같이 실습하면서 기초적인 파이썬 언어 문법 습득하여 
       스스로 파이썬 고급 코딩을 수행할 기본 실력 배양 
   (3) 기술 강의보다 파이썬 코딩 실습 위주이므로 교육생 역시 
       적극적인 실습 참여 의지가 필요함.                     
3. 교재 : 파이썬 자동화 교과서, 제이펍 출판, 구지라 히코즈쿠메 지음 
4. 실습 소프트웨어
    - MS Visual Studio Code (무료 제공, 1일차 설치 실습)
    - MS Terminal (무료 제공, 1일차 설치 실습)
    - Python IDLE Shell, Editor (무료 제공, 1일차 설치 실습)
