1-2. Docker Engine 설치 
	   	
	1) sudo apt-get remove docker docker-engine docker.io containerd runc
	   # 기존 도커 모듈 삭제 
	2) sudo apt-get update
	   # 우분투 업데이트
	3) sudo apt-get install \
    	   ca-certificates \
    	   curl \
    	   gnupg \
           lsb-release
	   # 도커 엔진 설치에 필요한 ca 인증서 서비스, curl, 오픈 프라이버시 가드(pgp 대용), lsb-release(리눅스 기본 규격)설치 
	4) sudo mkdir -p /etc/apt/keyrings
 	    curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /etc/apt/keyrings/docker.gpg
	   # 도커 공식 GPG Key 설정 
	5) echo \
  	   "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.gpg] https://download.docker.com/linux/ubuntu \
           $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null
	   # 도커 다운로드할 리포지터리를 설정함. 
	6) sudo apt-get update 
	   # 리포지터리 설정 후 업데이트 수행
	7) sudo apt-get install docker-ce docker-ce-cli containerd.io docker-compose-plugin 
           # 도커 엔진, containerd, Docker Compose 설치 
	
	8) sudo usermod -a -G docker $USER 
	   # docker group에 로그인한 유저를 포함함. 
	   # 재부팅 필요 

         공식 설치 가이드 문서 > https://docs.docker.com/engine/install/ubuntu/

2. NginX 웹 서버 설치 및 기동 
	1) docker pull nginx 
	2) docker image ls 
	3) docker container run --name websrv -d -p 80:80 nginx 
	4) curl localhost
	5) 웹 브라우저에서 확인 
	6) docker container ls 
	7) docker container stop websrv
	8) docker container start websrv 
	9) docker ps				#동작 중인 컨테이너 확인 
	   docker ps -a				#동작하지 않는 컨테이너까지 확인 
	10) docker rm [컨테이너id] or [name]	#컨테이너 삭제
	11) docker image ls or docker images 
	    docker images -a			#숨겨진 컨테이너도 확인
	12) docker rmi [이미지id] or [repository]

3. MySql db 서버 설치 및 기동 
   > MariaDB : MySQL이 Sun에 팔린후 MySQL 개발자들이 개발한 오픈소스 DB 
   > Oracle은 사용 DBMS 최강자이고, 저작권 분쟁으로 유명한 회사이므로 MySQL이 
      오픈소스로 지속하기 어려울 수 있어, MySQL 개발자들 MariaDB를 오픈소스 DBMS로 개발함. 
   	
	1) docker pull mysql 
	2) docker image ls 
	3) docker run --name mysql -e MYSQL_ROOT_PASSWORD=1234 -d -p 3306:3306 mysql:lastest
	4) docker ps -a
	5) docker stop mysql 
	6) docker start mysql 
	7) docker exec -i -t mysql bash 
           # 도커 엔진 위에 mysql 이미지 상의 bash 쉘에 접속, 인터랙티브 모드로 커맨드 실행 
	8) mysql -u root -p 
           Enter password:1234
